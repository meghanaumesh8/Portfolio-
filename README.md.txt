# Mental Healthcare Support Chatbot

## 📌 Overview
The Mental Healthcare Support Chatbot is an AI-powered system designed to provide emotional support, motivation, and guidance through natural conversation.

It helps users dealing with stress, anxiety, loneliness, and emotional challenges by offering a safe and non-judgmental space.

## 🚀 Features
- 🤖 AI-based conversational chatbot
- 💬 Empathetic and supportive responses
- 🌟 Positive affirmations & motivational quotes
- 🚨 Emergency helpline suggestions
- 🔒 Privacy-focused (no long-term data storage)
- ⚡ Fast and lightweight (works offline)

## 🛠 Tech Stack
- Python
- Streamlit (UI)
- Phi-3 Model (via Ollama)
- Natural Language Processing (NLP)

## ⚙️ How It Works
1. User enters message
2. NLP model understands emotion
3. AI generates supportive response
4. System provides motivation / guidance
5. Output displayed in chatbot UI

## 🎯 Objectives
- Provide emotional support
- Encourage positive thinking
- Offer crisis guidance
- Maintain user privacy

## 📊 Advantages
- 24/7 support
- Works offline
- Easy to use
- Low cost

## ⚠️ Limitations
- Cannot replace professional therapists
- Limited emotional understanding
- No long-term memory

## 🔮 Future Scope
- Voice interaction
- Emotion detection
- Integration with counselors
- Wearable device support
